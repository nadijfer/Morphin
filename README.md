# Morphin': Morphology Fixin'

**Morphin'** adalah bagian dari _final project_ mata kuliah Pemrosesan Bahasa Alami (Natural Language Processing; NLP) kelas B sebagai bagian untuk mengembangkan Grammarly untuk versi bahasa Indonesia. Proyek ini dikembangkan oleh kelompok **Never Like Programming (NLP)** dengan fokus masalah **perbaikan morfologi** yang mencakup:

* Kesalahan penggunaan imbuhan (awalan, akhiran, sisipan, konfiks)
* Kesalahan penulisan kata depan (di-, ke-, dari)
* Kesalahan bentukan kata dengan konfiks
* Kesalahan reduplikasi kata

Contoh: "mengdapatkan" → "mendapatkan", "didalam" → "di dalam", "mempertanggung jawabkan" → "mempertanggungjawabkan"
